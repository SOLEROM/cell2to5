OCT3032W | Octasic

Low-power, High-performance SoC for 4G LTE and 5G

The OCT3032W is a very low-power, high-performance System-on-Chip (SoC), with the industry’s highest processing capacity per watt for mission-critical wireless applications including 2G to 5G small cells, femtocells, network-in-a-box (NiB) applications, private wireless networks, tactical communications, public safety and Industrial Internet of Things (IIoT) networks.

The OCT3032W can run a complete on-chip base station or network-on-a-chip, including running the 3GPP protocol stacks and core network software. A single OCT3032W device can support up to four independent simultaneous software defined radio (SDR) air interfaces on a single chip, allowing OEMs to develop 2G to 5G capabilities on this one device.

The OCT3032W incorporates Octasic’s third-generation digital signal processing (DSP) architecture. It combines 32 ultra-low power DSP cores, an embedded quad-core Arm® Cortex®-A7 processor, embedded wireless hardware accelerators, a video encoder/decoder engine, low latency interconnects, and all necessary interfaces for a diverse range of applications.

High Performance

The OCT3032W is capable of up to 800/400 Mbps downlink/uplink throughput for LTE Advanced and LTE Advanced Pro. The SoC can also support 5G New Radio (NR) for both Non-Standalone (NSA) and Standalone (SA) deployments with throughputs up to 300/300 Mbps for downlink/uplink.

Low Power

The SoC implements power islands that can scale power consumption from 0.5W to 8W, depending on the application. Because the OCT3032W can be configured to supply power to only the components in use, it is an ideal choice for both compact handheld products and carrier-grade multi-radio applications.

Flexible Dual-mode Operation

The OCT3032W can operate as either a base station or as a user equipment (UE). By using the same component on both sides of the air interface, OEMs can simplify and accelerate application development, especially for applications like private radio networks with a focus on security, defense and industrial IoT.

Clock-less Architecture

Octasic’s Opus self-clocking asynchronous technology makes the OCT3032W the highest efficiency fixed-point processing device in the industry. Opus3, Octasic’s third-generation architecture, is software-compatible with Octasic’s field-proven Opus2 architecture, and adds 128-bit vector processing through its Core-Integrated Hardware Accelerator (CIHA).

Development Environment

Opus Studio, Octasic’s integrated development environment (IDE), offers a complete set of development tools for OCT3032 devices, including tools for code editing, compiling, simulation, debugging, and profiling. Octasic also provides a real-time kernel and RTOS services that can be used to rapidly develop embedded applications. Learn more about Opus Studio.

Physical Layer (PHY) Software

The OCT3032W is fully integrated with Octasic’s proprietary flexiPHY software supporting all 2G through 4G 3GPP Layer 1, or physical layer (PHY) releases to provide multi-standard cellular capabilities to developers, with development of a 5G PHY well under way. OEMs can leverage Octasic’s ready-to-deploy PHY software to quickly develop small cell base stations ranging from indoor femtocells to outdoor microcells. Learn more about flexiPHY.

OctNode Platforms

OctNode platforms are Octasic’s family of SDR base station subsystems that support any frequency band from 400 MHz – 6 GHz for any combination of GSM, CDMA2000, UMTS, LTE-FDD/TDD, 5G, or custom waveforms. OctNode platforms are offered in single/dual/quad-sector configurations and are powered by the OCT3032W SoC. OEMs can leverage OctNode platforms for their compact size, low power consumption, high processing power, video processing, and exceptional frequency agility. Its suite of integrated software reduces development time, cost, and risk, while being able to scale product lines as needed. OEMs can integrate their own Layer 2/3 stack or implement fully custom waveforms using Octasic’s Opus Studio development environment.

OCT3032W-based OctNode evaluation and development platforms will become available to customers starting in Q2 2019.
