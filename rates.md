# Rates

```
Mbps : Megabit per second (Mbit/s or Mb/s)
MB/s : Megabyte per second
1 byte = 8 bits
1 bit  = (1/8) bytes
1 bit  = 0.125 bytes
1 megabyte = 10002 bytes
1 megabit  = 10002 bits
1 megabit  = 0.125 megabytes
1 megabit/second = 0.125 megabytes/second

!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!
	1 Mbps = 0.125 MB/s
!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!
```

```
1   Mbps	0.125 MB/s
10  Mbps	1.25 MB/s
50  Mbps	6.25 MB/s
100 Mbps	12.5 MB/s
150 Mbps	18.75 MB/s
```
